# Projeto de Teste - GitHub Copilot AI Agent

## Objetivo

Este projeto Laravel foi criado com o objetivo de testar as ferramentas e capacidades do agente de IA do GitHub Copilot.

## Funcionalidades Implementadas

- **Calculadora de Soma**: Formulário para somar dois números
- **Calculadora de Subtração**: Formulário para subtrair dois números

## Estrutura do Projeto

O projeto segue a arquitetura MVC do Laravel com separação de responsabilidades:

- **Controllers**: `SomarController` e `SubtrairController`
- **Services**: Camada de serviços para lógica de negócio
- **Views**: Formulários em Blade para as operações matemáticas
- **Routes**: Rotas GET e POST para cada funcionalidade

## Tecnologias

- Laravel Framework
- PHP
- Blade Templates
- HTML/CSS

---

*Este projeto foi desenvolvido exclusivamente para demonstrar e testar as capacidades de automação de código do GitHub Copilot AI Agent.*
